Diabetes and Cerebrovascular Disease Analysis

Objective:

Effect of Diabetes on cerebrovascular disease in elderly patients .

Introduction

    Diabetes is a medical condition where the body struggles to regulate blood sugar (glucose) levels. This can happen for two main reasons: either the body doesn't produce enough insulin (a hormone that helps control blood sugar) or the cells become resistant to insulin, preventing it from working properly. When blood sugar levels remain too high over time, it can lead to various health problems.

Data Cleaning

  • Utilized Power BI's Power Query feature to efficiently clean and transform the dataset directly within the interface..
  • Removed rows containing null values using the "Remove Blank Rows" option to improve data quality.
  • Applied data standardization techniques to ensure consistency in variables and data entry formats..
  • Addressed missing values by calculating them using Power Query language, ensuring accurate data transformation. The "Visit 8 Age" column, which was initially miscalculated, was corrected using data from the "Visit 2" column.
  • Enhanced data accuracy by recalculating BMI with a DAX function to correct a user error in the dataset..
  • Improved dataset relevance by removing redundant columns such as DM, Non-DM, and STROKE, as their information was already captured in the Group column..
  • Data accuracy was improved by recalculating BMI due to a user error in the dataset.

Data Modelling

    In Power BI and Power Query, data was structured and categorized into logical tables such as Demographics, Labs, Cognitive Test, Diabetic Retinopathy, MRI, Walking Test, and Patient History to enhance both analysis and performance. A Relational Data Model (RDM) was created using these smaller logical tables, improving data organization. To facilitate seamless data integration, a surrogate key called ID was generated and added to all tables. Relationships between tables were then established using this ID key, enabling efficient data merging and analysis. By implementing the RDM, data integrity was significantly improved, ensuring accuracy and consistency. Leveraging this structured model in Power BI allows for intuitive visualizations and insightful data analysis, supporting informed decision-making.

Insights

  • Performed descriptive and statistical analysis, revealing a notable finding: individuals with diabetes exhibited a 38% higher likelihood of developing dementia..
  • Analyzed patient data and determined that 96% of patients with a history of Diabetes for more than 15 years, develop Hypertension.
  • Conducted correlation analysis and identified a strong positive correlation between Reverse Dipping Blood Pressure and poor diabetes management, thus increasing the incidence of cardiovascular disease and an elevated 75% risk for cognitive impairment.
